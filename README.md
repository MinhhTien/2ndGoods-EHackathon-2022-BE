## How to use
Install __Nodejs__

  * ```npm install``` to install dependencies
  * ```npm run build``` to build .ts to .js.
  * ```npm run init-database``` to synchronize the database with the entities.
  * ```npm run start``` to start.
  * ```npm run dev``` to run with _nodemon_.
